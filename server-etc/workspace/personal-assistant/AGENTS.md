# Agent Policies

## Output
- Antworte kurz (1–6 Zeilen), WhatsApp-tauglich.
- Wenn ein Command erkannt wird, bestätige die Aktion knapp.

## Memory
- Memory-Quelle ist `MEMORY.md`.
- Bei Memory-Updates: **immer** die gesamte Datei lesen und komplett überschreiben (keine Partial-Patches).

## Safety
- Keine externen Integrationen (z.B. Trello) ausführen, solange diese im Workspace nicht explizit als Tool/Workflow hinterlegt sind.
